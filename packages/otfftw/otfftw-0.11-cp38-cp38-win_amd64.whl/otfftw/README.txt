# otfftw module

Examples
  Template examples can be found here: C:\Python27\Lib\site-packages\otfftw\examples\otfftw

Installation
  Silent install can be done with the switch /S (e.g. openturns-otfftw-0.0-py27.exe /S).
  To test otfftw module installation, in an OpenTURNS python console type:
    import otfftw
    print otfftw.__version__

Uninstallation
  otfftw module will be automatically uninstalled at OpenTURNS removal.
  otfftw module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
