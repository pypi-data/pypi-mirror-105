# otrobopt module

Installation
  Silent install can be done with the switch /S (e.g. openturns-otrobopt-0.0-py27.exe /S).
  To test otrobopt module installation, in an OpenTURNS python console type:
    import otrobopt
    print otrobopt.__version__

Uninstallation
  OTRobOpt module will be automatically uninstalled at OpenTURNS removal.
  OTRobOpt module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
